Calculadora de Comisiones de Ventas Internacionales
Este proyecto es una solución empresarial diseñada para automatizar el cálculo de comisiones de vendedores según el país de operación, implementando una Arquitectura Limpia (Clean Architecture) y priorizando la eficiencia computacional y la precisión financiera.

Arquitectura del Sistema
La solución sigue el Modelo C4 y los principios de Onion Architecture, asegurando que el núcleo del negocio (Domain) sea independiente de los detalles de implementación (Infrastructure) y de los mecanismos de entrega (API/WebUI).
